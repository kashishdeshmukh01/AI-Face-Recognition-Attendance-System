# AI Face Recognition Attendance System

An attendance management system using Face Recognition and OpenCV.

## Features
- Face Detection using Haar Cascade
- Face Recognition using LBPH
- Automatic Attendance Marking
- GUI using Tkinter
- Attendance stored in CSV

## Technologies Used
- Python
- OpenCV
- NumPy
- Pandas
- Tkinter

## Project Structure
```
dataset/
trainer/
capture_faces.py
train.py
recognizer.py
gui.py
attendance.csv
```

## How to Run
```bash
pip install -r requirements.txt
python gui.py
```
